MyFlix API

The myFlix API is a RESTful backend service designed to support the myFlix web application, offering users access to detailed movie information, including data on films, directors, and genres. Built with Node.js and Express, and utilizing MongoDB for data storage, the API ensures efficient and scalable management of movie data. It features secure user authentication and profile management, along with the ability to curate a personal list of favorite movies.

MyFlix Client: Angular

MyFlixAngularClient is a client-side application built with Angular, allowing movie enthusiasts to explore a catalog of movies, directors, and genres. It includes features for user authentication and profile management, letting users save their favorite movies. The app demonstrates proficiency in Angular's component-based architecture and state management, while providing a responsive, single-page application experience with Angular Material.

MyFlix Client: React

The MyFlix React Client is a dynamic web application that lets users explore a catalog of movies, including detailed information on actors, directors, and genres. Built with React and Redux, it supports search, filtering, and a personalized list of favorite movies. The application integrates with the MyFlix REST API and is styled using Bootstrap, ensuring a responsive and engaging user experience.

Meet App

The Meet App is a serverless Progressive Web App (PWA) developed with React and integrated with the Google Calendar API. It offers features such as event filtering, offline functionality, and data visualization through charts. Leveraging serverless functions with AWS Lambda and adhering to PWA standards, the app provides a robust, cross-browser compatible solution with high test coverage.

Chat App

The Chat App is a mobile application developed using React Native and Expo, allowing users to send text messages, share images, and send their location. With features for online and offline communication, it integrates Firebase for authentication and real-time data storage. The app includes customizability options and leverages the Gifted Chat library for an interactive chat experience.
